# 🎮 Video Game Sales Analysis

This project explores global video game sales data to uncover patterns in platform performance, genre popularity, and user/critic ratings.

## 📊 Objective
Analyze historical video game data to identify trends and factors that influence sales across different regions.

## 🔍 Key Insights
- Differences in sales performance across platforms (Wii, PS3, X360, etc.)
- Genre popularity trends over time
- Relationship between critic scores and user scores
- Regional sales distribution (NA, EU, JP)

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- SciPy

## 📁 Dataset
The dataset includes:
- Game titles
- Platforms
- Release year
- Genre
- Sales (NA, EU, JP, Other)
- Critic and user scores
